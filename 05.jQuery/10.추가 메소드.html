** each 메소드

배열의 모든 인덱스에 순차적으로 접근하고자 할떄
객체가 가지고 있는 모든 속성에 순차적으로 접근하고자 할 떄
for in문과 유사하게 수행되는 메소드 

        1. $.each(객체|배열, function([매개변수1, 매개변수2] {
              순차적으로 접근할 때 마다 실행할 소스;   
           }))
        
        2. 객체 배열.each(function() {
              순차적으로 접근할 때마다 실행할 소스;
           })

        만약 갹체를 제시했다면
        천번째 매개변수에는 순차적으로 접근할 때 마다 객체의 속성명(키)이 담김
        두번째 매개변수에는 해당 속성값이 담김
